Domain Name
A domain name represents one or more IP addresses. The domain name is comprised of all characters in a website’s URL that follow https:// and precede the first /.

Why Domain Names are Important

Domain names are easier for humans to remember than IP addresses, which are strings of numbers computers interpret to locate a website.

Why Domain Names are Unique

All domains are subdomains of another domain. Top-level domains include extensions such as .com and .org, and are the portion of domain names that can be used an unlimited number of times. The first subdomain of a top-level domain will be unique because it belongs to the portion of the domain name that identifies a specific website.

Registering a Domain Name

To register a domain name, one must acquire one from an accredited registrar. In exchange for a sum of money and contact information, a registrar can provide an unclaimed domain name. The owner then has the right to create a website under said domain name for a finite span of time.

Aquiring Domain Names Already in Use

If one wishes to acquire a domain name already in use, they can buy it from the owner, or wait for the owner’s lease on the domain name to expire and hope that it isn’t renewed.
