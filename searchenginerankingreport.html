Search engine report

How does a search engine work?
Before a search engine can catalogue an index of webpages, it needs to rely on webcrawlers. Webcrawlers will download a webpage and then follow the links referenced thereon to new pages. In this way, webcrawler programs crawl from one website to another.
A popular algorithm search engines use to rank the importance of the sites webcrawlers record is the Page Rank Algorithm, as developed by Google. According to this algorithm, the number of external links that lead back to a site most significantly affects its placement in search results.
Why do different search-engines give different ranking to websites for the same search phrase?
Different search engines rely on different sorting algorithms that value different properties. For instance, some search engines, such as Google, place less value on specific keywords and small phrases than on trying to interpret the full context of what is input into the search bar. A search engine like Bing is more likely to treat the keywords comprising a single query as separate, and target them specifically.
What is a <meta /> tag?
Meta data is information that is readable by machines but that usually doesn’t display in the browser page itself. The <meta /> tag is always placed within the of a webpage. <meta name=""/> is used to input information such as keywords, descriptions, and author name. Other attributes can be used to specify things like character sets, or how browsers should display the page.
What is the purpose for Keyword and Description meta tags?
Meta keywords can be included in html code through the use <meta name="keyword"/>. Historically, some website creators have taken advantage of algorithms that favor meta keywords by including long lists of popular keywords, even if they relevant to their page’s actual content, in order to divert extra traffic to their site. The original point of keyword tags was to make it easier for users to find relevant content related to their search query, and for the websites whose content reflected said tags to attract interested users.
A meta description is a specific attribute of the meta tag that allows creators to write a descriptive blurb surmising their site’s content. This description may or may not be displayed in search engines in favor of text pulled from their page’s main body. Meta descriptions are defined by meta followed name=”’. The description is then specified in the content=”” that follows.
The purpose of a meta description is to allow creators to present the description of their website as they want users to see it when scrolling through search results.
Why might you include meta tags in your pages?
Meta data is handled differently between web browsers. Both meta keywords and meta description tags may help search engines categorize websites, but aren’t as impactful in website ranking as they used to be.
Meta descriptions make it easier for a web browser to sort websites, and may also make it easier for the user when parsing through search results. A Meta description tag can potentially be more concise and cohesive than what is take from your content by default, though Google may ignore your description in favor of this anyway.
Do meta tags make your pages more easily searchable or improve its ranking?
Meta keyword tags are no longer as universally valued as they once were. In Response to their popular abuse, Google officially changed its policy so that meta keywords no longer weigh heavily in their ranking process. However, even in contexts where they don’t heavily skew results on their own, it’s possible they can contribute to search rankings in some small cumulative way. Including keywords in html code generally won’t hurt search results as long as there aren't to many and they are relevant to the site's content. If one intends to include meta tags, it may make a small difference to consult a ranking list and check for the most popular keywords that are also relevant one’s site.
Meta tags such as descriptions and titles have a greater impact on search results. Keywords that appear in meta descriptions will not directly effect ratings within the Google search engine, however, keywords types into the search bar that happen to match the description will be bolded for users, and can in that way attract more traffic.
The title tag is the most impactful meta tag when it comes to determining search ranking.
